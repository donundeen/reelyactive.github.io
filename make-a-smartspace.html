Make a Smart Space

Here we'll present how to create a Smart Space that you can visualise in a web browser like this. All you'll need is two of our free software packages and some of our hardware. The image below highlights the reelyActive smartspaces stack.

In a nutshell, you'll install and run smartspaces and hlc-server on a server and connect some hardware to provide the source data. A few simple configurations and you have a live Smart Space!
