    Highlight key information in a Time Series visualization.

    Our step-by-step guide to highlight key information or anomalies in a Time Series visualization.

    The TL;DR (Too Long; Didn't Read)

    Learn how we at reelyActive use Kibana annotations in TSVB visualization to observe specific behavior of assets/persons.

    What will this accomplish?
    An annotated visualization which highlights specific information like the door opening.
    Is there an easier way?
    Manual counts and analysis of asset/person behavior using pen and paper?
    So why would I read this?
    To learn both how and why to annotate specific behavior in a TSVB visualization.

    Prerequisites

    reelyActive open source software with Elasticsearch and Kibana.

    Choosing TSVB visualization   Step 1 of 2

    Choose a TSVB visualization and add annotations to make reading information more efficient.

    Why a TSVB visualization?
    The annotations feature is only available in Time Series Visual Builder (TSBV) visualization in Kibana.
    Why Kibana?
    Kibana makes it easy to visualise data from an Elasticsearch database, where the source data is stored.

    Open Kibana and then:

    1. Select a TSVB visualization from Visualize tab or Dashboard tab
    2. Click the gear icon
    3. Select Edit visualization

    The TSVB visualization will open on the Metrics tab. The next step will be to define a meaningful annotation.

    Adding annotations   Step 2 of 2

    Define a meaningful annotation by filtering data and add it to the visualization

    Why annotate?
    To help readers track information and deepen their understanding of visualization.
    What's filtering data?
    This means the data sets are refined by removing data that can be repetitive or irrelevant.

    From the TSVB visualization:

    1. Click the third tab: Annotations from the main page
    2. Click the Add data source button

    From the Annotations tab:

    1. Select your Index pattern
    2. Choose timestamp in the Time field
    3. Enter your query string in the Search area, by clicking in the Search area kibana suggests filters
    4. Select an icon that will be display each time the event will appear
    5. Enter the Field that corresponds to your query
    6. Enter a text which will be displayed each time the mouse is over an event.
