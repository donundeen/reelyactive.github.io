reelyActive technology is a platform for creating Smart Spaces. A Smart Space understands who/what is present. This opens up endless possibilities for creating responsive environments that react to real-world context.

Welcome to diyActive, our Do-It-Yourself page, which is intended to help you explore these endless possibilities yourself.
