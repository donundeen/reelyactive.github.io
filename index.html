reelyActive technology is a platform for creating Smart Spaces. A Smart Space understands who/what is present. This opens up endless possibilities for creating responsive environments that react to real-world context.

Welcome to diyActive, our Do-It-Yourself page, which is intended to help you explore these endless possibilities yourself through the projects and tutorials below, all of which are based on our open source code on GitHub and npmjs.

Browse reelyActive hardware and our open source software via:

Our active projects include:

  • barnowl, our Internet of Things middleware package
  • barnacles, which efficiently propagate contextual information from a local to a global scale
  • hlc-server, which builds on barnowl & barnacles to serve hyperlocal context
  • smartspaces-server and smartspaces-client, together as smartspaces, which visualise hyperlocal context in a web browser
  • json-silo, for hosting digital stories for smartspaces
  • midiot, where MIDI meets the IoT for live art installations
  • barnowl-test for out-of-the box hardware testing
  • locationdemo to easily test indoor location performance
